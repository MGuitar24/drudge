# drudge
A CLI that prints the drudge page headlines to stdout 

```
Usage: drudge [options]
Options:
  -v        Print the version
  -s        Print the output as a string
  -h        Print this help menu
  -w [n]    Watch for updates every n seconds (default is 30 seconds)
  ```
